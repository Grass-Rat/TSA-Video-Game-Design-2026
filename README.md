This project is a JavaScript application built with Phaser that demonstrates multi-level game design and object-oriented programming.

# PROJECT STRUCTURE

```
-/assets
-/src
  /scenes
  /objects
  main.js
```

 # INSTALLATION
Clone the repo
```
git clone https://github.com/Grass-rat/TSA-Video-Game-Design-2026.git
cd TSA-Video-Game-Design-2026
```
Install Dependencies
```
npm install
```
